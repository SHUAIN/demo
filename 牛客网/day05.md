# 0.2+0.4=0.6000000000001
>这是因为二进制转换的问题导致
>解决
```js
let a = 0.2,
    b = 0.4;
// toFixed() 方法可把 Number 四舍五入为指定小数位数的数字。返回字符串类型
console.log(parseFloat((a + b).toFixed(1)));
// 扩展
// toPrecision 把数字格式化为指数
```
